# WeChat Smoke Checklist

## DevTools

1. Import `build/wechatgame-debug`.
2. Confirm the starter scene is visible.
3. Confirm the confirm button renders and responds.
4. Confirm there is no startup failure log.

## Device

1. Install the latest package on device.
2. Confirm the starter scene is visible.
3. Confirm the button responds.
4. Confirm safe-area padding looks correct.
