# WeChat Smoke Checklist

## DevTools

1. Import the latest `build/wechatgame-debug` directory.
2. Confirm startup reaches the first visible scene.
3. Confirm the simulator is not black.
4. Confirm input works in the first scene.
5. Confirm no startup failure logs remain.

## Device

1. Install the latest debug or release package on a device.
2. Confirm startup reaches the first visible scene.
3. Confirm input works.
4. Confirm safe-area layout is sane.
5. Record result as `verified`, `partial`, or `blocked`.
