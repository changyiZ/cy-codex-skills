# Godot LLM Routing Index

Source-of-truth rules:
1. Official Godot docs and official demo projects come first.
2. If community posts conflict with official sources, trust the official sources.
3. If the repo is pinned to a different Godot version, respect the repo version over generic stable advice.
4. Use stable docs by default, and use latest docs only for unreleased or recently changed behavior.
5. Demo projects are for pattern discovery; confirm final API details in the docs or class reference.

## Version Policy

- Stable release baseline: `Godot 4.6.1-stable` -> https://godotengine.org/download/archive/4.6.1-stable/
- Stable docs root: https://docs.godotengine.org/en/stable/
- Latest docs root: https://docs.godotengine.org/en/latest/
- Release policy: https://docs.godotengine.org/en/stable/about/release_policy.html
